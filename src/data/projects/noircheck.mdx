# NoirCheck - Digital Content Authenticity Verification Platform

## Descripción

NoirCheck es una plataforma innovadora para la verificación de autenticidad de contenido digital, que combate la desinformación usando tecnología blockchain y zkTLS. Incluye aplicación móvil y web, sistema de estadísticas en tiempo real y compatibilidad con wallets XION y MetaMask.

## Características principales

- Subida y verificación de archivos (fotos, documentos, etc.)
- Registro de contenido en blockchain con atribución de autor
- Verificación instantánea y scoring de confianza
- Estadísticas y actividad en tiempo real
- Integración con wallets (XION, MetaMask, WalletConnect)
- Interfaz moderna y responsive
- Persistencia local de datos y actividad
- Multiplataforma: iOS, Android y web

## Tecnologías

- Python + FastAPI (backend)
- Next.js 15 + React 19 (frontend web)
- React Native + Expo (app móvil)
- Tailwind CSS, Material Design 3
- Blockchain XION zkTLS

## Estado

Proyecto en desarrollo y operativo, con migración completa a Next.js y React Native, y nuevas funciones de estadísticas y wallets.
