# NoirCheck - Digital Content Authenticity Verification Platform

## Description

NoirCheck is an innovative platform for digital content authenticity verification, combating misinformation using blockchain and zkTLS technology. It includes mobile and web applications, a real-time statistics system, and compatibility with XION and MetaMask wallets.

## Main Features

- **File Upload and Verification**: Support for photos, documents, and more.
- **Blockchain Registration**: Content logging with author attribution.
- **Instant Verification**: Real-time scoring and trust indicators.
- **Real-Time Activity**: Dashboard with live statistics and activity feeds.
- **Wallet Integration**: Support for XION, MetaMask, and WalletConnect.
- **Modern UI**: Fully responsive and intuitive interface.
- **Local Persistence**: Offline-first approach for local data storage.
- **Multi-Platform**: Available for iOS, Android, and Web.

## Technologies

- **Python + FastAPI**: For the high-performance backend.
- **Next.js 15 + React 19**: Modern frontend architecture.
- **React Native + Expo**: Cross-platform mobile development.
- **Tailwind CSS + Material Design 3**: Consistent and beautiful design system.
- **Blockchain XION zkTLS**: Decentralized trust layer.

## Status

The project is currently in active development, with a complete migration to Next.js and React Native, and new features for statistics and wallet integration.
