# Chuches Inventario - Professional Inventory Management

## Description

Professional inventory management system for businesses, featuring a modern interface and robust features for tracking stock across multiple platforms.

## Key Features

- **Cross-platform**: Support for Android and Desktop (Windows/Linux/macOS).
- **OCR Scanning**: Use the device camera to scan barcodes and product information.
- **Real-time Inventory**: Keep track of stock levels and receive low-stock alerts.
- **Reporting**: Generate professional PDF reports for inventory status and sales.
- **User Permissions**: Robust role-based access control.

## Technologies

- **Python**: Core logic and backend.
- **Kivy / KivyMD**: Modern UI framework for cross-platform apps.
- **Buildozer**: Tool for packaging Python apps for Android.
- **Cython**: Performance optimization.
- **Pytest**: Comprehensive unit testing.

## Status

Project completed and operational, used for real-world inventory tracking in small businesses.
