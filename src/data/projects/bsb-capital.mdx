## Project Description

A comprehensive financial services platform specializing in cryptocurrencies and investments. It includes a membership system, personalized consulting, analysis tools (BSB Maker), an educational academy, and an administrative dashboard with a modern interface and interactive animations.

## Main Features

- **Modern UI & Animations**: Built with Framer Motion for a premium feel.
- **Membership & Roles**: Granular access control for different user tiers.
- **Personalized Consulting**: Dynamic forms and appointment scheduling.
- **Financial Analysis Tools**: Proprietary "BSB Maker" for advanced analytics.
- **Educational Academy**: Premium educational content and courses.
- **Admin Dashboard**: Full control over users, payments, and content.
- **Payment Integration**: Seamless checkout via Stripe and Hotmart.
- **Security**: Robust protection using Supabase Auth and RLS.
- **Transactional Emails**: Automated notifications via Resend.

## Technologies and Tools

- **React 18 + Vite + TypeScript**: Core frontend stack.
- **Tailwind CSS + shadcn/ui**: Modern styling and component library.
- **Framer Motion**: Advanced animations and micro-interactions.
- **Supabase**: Backend-as-a-Service (PostgreSQL, Auth, RLS).
- **Stripe / Hotmart API / Resend**: External service integrations.
