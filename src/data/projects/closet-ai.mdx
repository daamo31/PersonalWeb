# ClosetAI - Your AI-Powered Smart Wardrobe 👗✨

## Project Description

ClosetAI is a mobile-first web application that digitizes your wardrobe and uses artificial intelligence to suggest outfits based on the current weather, your personal style preferences, and the garments you own. It includes a Telegram bot for quick access and a virtual try-on feature so you can see how clothes look before wearing them.

## Main Features

- **Smart Outfit Generator**: AI-powered suggestions combining weather data and your wardrobe.
- **Digital Wardrobe**: Add, organize, and categorize all your garments with photos.
- **Virtual Try-On**: Visualize outfits on a virtual model using AI.
- **Weather Integration**: Real-time weather data for location-aware outfit recommendations.
- **Telegram Bot**: Access your wardrobe and get outfit suggestions directly from Telegram.
- **Plan Management**: Freemium model with garment capacity limits and plan upgrades.
- **Quick Actions**: Fast access to wardrobe, outfit generation, and weather info from the home screen.

## Technologies and Tools

- **Next.js 15 + TypeScript**: Modern full-stack framework with server-side rendering.
- **Tailwind CSS + shadcn/ui**: Clean and responsive mobile-first UI.
- **Supabase**: Backend-as-a-Service with PostgreSQL, Auth, and file storage.
- **OpenAI API**: AI outfit generation and virtual try-on capabilities.
- **Telegram Bot API**: Remote wardrobe interaction via Telegram.
- **Weather API**: Location-based weather data for context-aware recommendations.
